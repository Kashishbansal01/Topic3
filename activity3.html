Control Interface

A perimeter with a set of controls that regulates the exchange of information across linked information systems and enforces security regulations.

User-friendly, Data Security, Two-Way Communication, and Manage Faults are high level aspects of the control interface. Occupancy Sensors (Ultrasonic, PIR, and IR), Light Sensors, and other elements are required (Photodiodes, Photoresistors, Photovoltaic). HVAC System, Building Management, and Security Control System are the users of the control interface, which is effectively managed by the security team.
